# Confluence Flow Pro

Confluence 极速下载助手 (Chrome Extension)

## 🎯 核心功能

- **🚀 极速导出当前页**：绕过 Confluence 原生复杂的菜单跳转，直接一键下载当前页面的 PDF。
- **📦 批量导出子文档**：智能扫描侧边栏目录树，一键将当前目录下所有子文档加入下载队列。
- **🛠 自动重命名**：自动提取页面标题作为文件名，并清洗非法字符，确保下载文件井然有序。
- **⏱ 智能排队**：批量下载时自动设置时间间隔，防止请求过快被服务器拦截。

## 🛠 安装方法

1. 下载本项目代码到本地。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载解压的扩展程序”**。
5. 选择本项目文件夹即可。

## 💡 使用说明

- 进入任何 Confluence 页面，右下角会自动出现操作按钮。
- 如果扫描到子文档，橘色按钮会自动显示可下载的数量。
- 下载文件将默认保存至浏览器的下载路径下的 `Confluence_Exports/` 文件夹中。
